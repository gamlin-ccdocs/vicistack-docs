## Key Takeaways

1. **VICIdial runs 14,000+ installations worldwide.** It's not a hobby project -- it's battle-tested production software powering call centers of every size, from 5-seat shops to 500+ agent operations.
2. **The Asterisk/MySQL/Apache stack is the foundation.** Understanding this architecture matters because it determines your hardware requirements, scaling approach, and troubleshooting workflow.
3. **VICIdial vs. hosted solutions is a trade-off, not a competition.** You trade monthly per-seat fees for upfront infrastructure and Linux admin work. If you have the technical chops, the ROI is massive.
4. **ViciBox is the fastest path to a working system.** The ISO install gets you [from bare metal to first](/blog/sip-registration-failed-fix/) campaign in under 2 hours. AlmaLinux scratch installs give more control but take longer.
5. **Scaling is multi-server, not bigger-server.** VICIdial clusters by separating web, telephony, and database roles across machines. Plan for this from day one if you're going past 50 seats.
